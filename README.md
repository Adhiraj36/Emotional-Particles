# Emotional-Particles
A 3D generative art project merging Three.js and MediaPipe. It tracks 468 facial landmarks to transform 20,000 particles in real-time.  Dynamic Morphing: Shapes shift based on mood (Happy, Angry, etc.).  Blink Detection: Uses EAR logic to trigger cosmic fireworks.  Tech: WebGL, BufferGeometry, and LERP transitions.
